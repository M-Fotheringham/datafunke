Get *funke* with data analysis.

## Installation
Open a command prompt and clone the repository locally with: <br>
```
git clone https://github.com/M-Fotheringham/datafunke
```
Then navigate into the repository: <br> 
```
cd datafunke
```
and pip install: <br>
```
pip install .
```
